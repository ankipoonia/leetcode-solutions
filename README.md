# LeetCode Solutions

This repository contains my solutions to various LeetCode problems, implemented in Python. LeetCode is a popular platform for practicing coding interview questions, covering topics like algorithms, data structures, and problem-solving techniques.

## Structure

- There are folders for different catagories of problems (e.g., Arrays, DP etc.).
- Each solution is organized in its own Python file, named after the problem (e.g., `two_sum.py`).
- Solutions are written in Python 3, focusing on clarity, efficiency and elegance.
- Comments in the code explain the approach and key steps.

## How to Use

1. Clone the repository: `git clone https://github.com/ankipoonia/leetcode-solutions.git`
2. Navigate to the directory: `cd leetcode-solutions`
3. Run any Python file: `python problem_name.py`
